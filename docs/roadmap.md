# Roadmap

## 0.1.0 Scaffold

- Project name, folder, catalog, docs, update manifest, preview UI, verification script, publish script.
- Tab5 ESP-IDF firmware shell and OTA partition plan.
- Public-safe mode contract for Meshtastic and MeshCore.

## 0.2.0 First Device Loop

- Tab5 boots with LVGL launcher.
- USB host detects C6L CDC serial.
- Meshtastic driver can read status, list nodes, and send a channel message.
- Update Center checks GitHub Pages manifest.
- Mic Deck records WAV to SD.
- IR Lab learns and replays one NEC code.

## 0.3.0 MeshCore Assist

- MeshCore command console.
- `start ota` helper.
- OTA AP/upload page handoff.
- Mode profile persistence.

## 0.4.0 T-Deck Companion

- Z-Deck/T-Deck status readout.
- Payload launch bridge.
- Safe OTA status and update guidance.
- No-LittleFS default preserved.

## 0.5.0 Field Suite

- Camera QR/device-label scan.
- IMU gesture shortcuts.
- RS485 console.
- Exportable redacted field report.
